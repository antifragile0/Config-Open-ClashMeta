### Setting Configuration DNS

- NOTES : `DNS Provider Khusus Nol Kuota`
  - `112.215.198.248` : DNS XL
  - `112.215.198.254` : DNS XL
- Silahkan Ganti Dengan DNS Provider / DNS Global
- Contoh :
  - `1.1.1.1` : Cloudflare
  - `8.8.8.8` : Google
  - `9.9.9.9` : Quad9
